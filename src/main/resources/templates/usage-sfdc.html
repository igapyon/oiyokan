How to connect to Oiyokan from Salesforce.

How to use Oiyokan OData server from Salesforce

This article describes how to connect to the Oiyokan OData server from Salesforce.

Find External Data Source

Find External Data Source using Quick Find, and click it.

Create New External Data Source

Click New External Data Source to create new External Data Source.

Setup New External Data Source

Fill input field like below. Type should be Salesforce Connect: OData 4.0. URL should be https://oiyokan.herokuapp.com/odata4.svc/. And click Save.

Click Validate and Sync

Click Validate and Sync button to validate odata settings. If you see a timeout message, try again later. It may be resolved.

Select EntitySet (Table)

Select some EntitySet (Table) from list want to connect and check them. And click Sync.

Validate EntitySet (Table)

Click Validate link to validate it.

Click Run Queries

Click Run Queries to execute validation.

Check result of validation

Check result of validation.

Fields and Relationships at Object Manager

Open object manager to check Fields and Relationships added by OData.

End of this article

With the above steps, you can use Oiyokan as a external object from Salesforc. Here is the end of this article. Thank you very much reading this article.
